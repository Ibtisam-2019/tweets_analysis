Cyberbulling Detection

Here we propose to use Twitter Account Analysis Service. This project will use cyber forensics, cyber-bullying, and cyber-harassment. It will create a web page which allows to input a twitter username in order analysis it. As a result, it illustrates full account name, date of account creation, date of last tweet, how many followers and following him/her. In addition, it creates selection to research about some words by using cyber forensics methodology
